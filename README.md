# Karaoke Web Application

A modern, interactive karaoke web application that allows users to sing along to their favorite YouTube karaoke tracks with a scoring system.

## Features

- **YouTube Integration**: Stream karaoke videos directly from YouTube
- **Song Queue**: View and manage upcoming songs
- **Search Functionality**: Quickly find your favorite songs
- **Responsive Design**: Works on desktop and mobile devices
- **Scoring System**: Get feedback on your performance
- **Modern UI**: Clean, intuitive interface with smooth animations

## Setup

1. Clone this repository
2. Open `index.html` in a modern web browser
3. Start singing!

## How to Use

1. Browse or search for songs in the list
2. Click on a song to start playing
3. Sing along with the lyrics (if available)
4. View your score after each song
5. Use the player controls to navigate between songs

## Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- YouTube IFrame Player API
- Google Fonts
- Font Awesome Icons

## Note

This application uses the YouTube IFrame Player API to embed and control YouTube videos. Make sure you have an internet connection to load the videos.
